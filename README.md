# Proyecto Flask - Ejercicios Académicos

## Descripción proyecto paso a paso
Aplicación web desarrollada con Flask que contiene dos ejercicios:

1. **Ejercicio 1**: Cálculo de promedio y estado académico
   - Ingreso de 3 notas (10-70)
   - Ingreso de asistencia (0-100%)
   - Cálculo de promedio y determinación de aprobación

2. **Ejercicio 2**: Determinación del nombre más largo
   - Ingreso de 3 nombres
   - Identificación del nombre con más caracteres
   - Mostrar longitud del nombre
